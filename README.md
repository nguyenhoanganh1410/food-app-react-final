# Food App Project

This project was developed with React.

Project Status : 100%

Live demo: https://foodapp-react.herokuapp.com/

### Used Technologies

* HTML
* CSS/SCSS
* React
* React Router
* Firsebase
* Json Server

### Install & Run

